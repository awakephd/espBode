Helper function to facilitate stream operations on DEBUG objects.

This function overrides the << operator to allow a DEBUG& object to be streamed to a Print& object. This allows the use of DEBUG methods that set a condition within the stream.

Examples:

Debug << Debug.Type(DEBUG::ERROR) << "Something bad happened!\n"
Debug << Debug.Progress() << "We're getting there!\n";

Parameters
pstreamReference to a Print object (destination of the stream operation)
debug_argA method that returns a reference to the DEBUG object
Returns
The debug_arg is passed through as the return object.
