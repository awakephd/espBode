◆ setup_WiFi()

void setup_WiFi ( )

Set up the WiFi connection.

The WiFi setup is separated out from the rest of the setup() routine just to keep the latter from being too cluttered. Note that this function blocks until the WiFi connection is successfully established; in other words, the ESP-01 will be completely unresponsive if no WiFi connection is made.
