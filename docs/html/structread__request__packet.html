Detailed Description

Structure of the VXI_11_DEV_READ request packet.

In addition to the basic RPC request data, the DEV_READ request includes the link id, request size (maximum amount of data to send per response), timeouts for lock and i/o, flags (can signal the use of a terminating character), and the terminating character if any.
