enum fy::wave_types

Enumeration of the basic wave form ids for the FeelTech FY-series AWGs.

For the FY-series of AWGs, wave form type is indicated by an integer, included as part of the write command or returned after a read command. For example, to set the AWG to output a Rectangle wave on the main channel, the following command will be send to the FY AWG over its USB/serial communication port: WMW2 (where 2 = the Rectangle id from the wave_types enumeration).
