Detailed Description

Declaration and definition of the siglent::BSWV_types enumeration.

This enumeration is actually not needed o4 used by the current espBode implementation. Since the Bode plot only uses a sine wave, any SCPI command for wave type is assumed to call for a sine wave. This enumeration is included in case of possible future extension of functionality.
