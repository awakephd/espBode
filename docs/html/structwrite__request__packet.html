Detailed Description

Structure of the VXI_11_DEV_WRITE request packet.

In addition to the basic RPC request data, the DEV_WRITE request includes the link id, timeouts for lock and i/o, flags (can signal the addition of an "end" character at the end of the data), length of the data being sent, and the data itself.
