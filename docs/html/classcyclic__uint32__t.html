Detailed Description

4-byte integer that cycles through a defined range.

The cyclic_uint32_t class allows storage, retrieval, and increment/decrement of a value that must cycle through a constrained range. When an increment or decrement exceeds the limits of the range, the value cycles to the opposite end of the range.
