◆ packet_buffer_sizes

enum packet_buffer_sizes

Enumeration of the sizes of the various packet buffers.

The buffers must allow sufficient space to receive the longest expected data for the type of packet involved.

Enumerator
UDP_READ_SIZE 

The UDP bind request should be 56 bytes.

UDP_SEND_SIZE 

The UDP bind response should be 28 bytes.

TCP_READ_SIZE 

The TCP bind request should be 56 bytes + 4 bytes for prefix.

TCP_SEND_SIZE 

The TCP bind response should be 28 bytes + 4 bytes for prefix.

VXI_READ_SIZE 

The VXI requests should never exceed 128 bytes, but extra allowed.

VXI_SEND_SIZE 

The VXI responses should never exceed 128 bytes, but extra allowed.
