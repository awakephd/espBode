Detailed Description

Structure of an RCP/VXI packet prefix.

TCP communication uses a 4-byte prefix followed by the variable- length packet. The prefix contains the packet length (not including the length of the prefix) in the lower 31 bits. The most significant bit is a flag signalling whether there is more data to come ("fragment" bit).
