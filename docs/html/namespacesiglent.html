enum siglent::BSWV_types

Enumeration of the BSWV wave types used by Siglent AWGs and oscilloscopes.

For Siglent instruments, wave form is expressed in an SCPI command by a string (e.g., "SINE") rather than by a number. It is very likely that the only one we care about is the sine wave. However, if it ever becomes necessary to translate the Siglent waveform into the waveform usedby one of the budget AWGs that espBode-2 covers, it will be helpful to have an integer code to represent the Siglent waveforms. This enumeration is based on the list provided in the WGEN section of the SDS Series Digital Oscilloscope Programming Guide EN11G. Note that we are only listing the BaSic WaVeform types; we are ignoring the ARbitrary WaVforms.
