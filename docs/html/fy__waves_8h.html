Detailed Description

Declaration and definition of the fy::wave_types enumeration.

This enumeration is not needed or used by the current espBode implementation. Since the Bode plot only uses a sine wave, any request to set wave type defaults to using 0 (Sine wav for the FY series). However, this enumeration is included for the sake of possible future extensions of functionality.
