◆ operator<<()

Print & operator<< ( Print &  stream,
cyclic_uint32_t &  value 
)
inline

Converts cyclic_uint32_t to uint32_t for streaming.

The following allows "transparent" use of a cyclic_uint32 in a stream operation without having to explicitly cast it; e.g., instead of having to write

Serial << (uint32_t)cyclic_uint32_t_instance; or
Serial << cyclic_uint32_instance();

we can just write

Serial << cyclic_uint32_val;
Parameters
streamReference to a Print object to which to stream the value
valueThe cyclic_uint32_t object to output to the stream
Returns
Reference to the Print object

A quick way to get a power of 10 up to +/-9.

Note that this function uses integer multiplication, then converts the result to double-precision floating point.

Parameters
pThe exponent to use (range: -9 to 9)
Returns
If -9 <= p <= 9, return 10^p; else return 1
